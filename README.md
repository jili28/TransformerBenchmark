# AFLT project
==============================

Experiments used for Dyck-(k, M) recognition task and benchmarking of Transformer performance on formal languages 
for the Advanced Formal Language Course at ETH Zurich.

## Project Organization

## Datasets
Available datasets to benchmark our model against the baselines: 

- Dyck-(k, M)
- Parity
- First

### Usage of repository 

Specify training hyperparameters in `sweep.yaml` and run `train.py`
